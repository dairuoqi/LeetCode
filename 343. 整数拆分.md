#### 343. 整数拆分

给定一个正整数 n，将其拆分为至少两个正整数的和，并使这些整数的乘积最大化。 返回你可以获得的最大乘积。
```
示例 1:

输入: 2
输出: 1
解释: 2 = 1 + 1, 1 × 1 = 1。

示例 2:
输入: 10
输出: 36
解释: 10 = 3 + 3 + 4, 3 × 3 × 4 = 36。

```
思路：自底向上的动态规划
* 假设n的最大化乘积为dp[n]
* x = max (dp[n-j],n-j),  y = max(dp[j],j)
* max (n) = max (x * y) 

```
int integerBreak(int n) {
    int *dp = (int *) calloc(n + 1,sizeof(int));
    dp[1] = 1;
    int x,y;
    for (int i = 2; i <= n; i++) {
        int rs = 1;
        for (int j = 1; j <= i/2; j++) {
            x = dp [i-j] > i - j ? dp[i-j] : i-j;
            y = dp[j] > j ? dp[j] : j;
            if ( rs < x * y ) rs = x * y;
        }
        dp[i] = rs;
    }
    return dp[n];
}
```
